## 💾 설치 방법

### 필요 조건
- PC (Windows) Steam 또는 Epic Games 버전
- 별도 프로그램 설치 불필요 (BepInEx 포함)

### 설치 순서

**1단계 - 다운로드**
아래 Releases에서 `TLSA-한글패치.zip` 다운로드

**2단계 - 압축 해제**
게임 설치 폴더에 **덮어쓰기**로 압축 해제
```
기본 경로: C:\Program Files (x86)\Steam\steamapps\common\The Last Stand Aftermath
```

**3단계 - 게임 실행**
Steam에서 정상 실행
> ⚠️ 첫 실행 시 BepInEx 초기화로 **1~2분 로딩이 길어질 수 있습니다** (정상)

## ❓ 자주 묻는 질문

**Q. 게임 업데이트 후 번역이 안 돼요**

A. 게임 업데이트 시 패치를 다시 설치해야 할 수 있습니다. Releases에서 최신 버전을 확인해주세요.

**Q. 일부 텍스트가 영어로 표시돼요**

A. 후원자 이름 등 일부 고유명사는 의도적으로 영어 유지합니다.

**Q. Epic Games 버전에서 오류가 나요 / 번역이 안 돼요**

A. 에픽 버전은 첫 실행 시 에픽 계정 연동 팝업이 뜰 수 있습니다. 연동을 완료하면 정상 작동합니다. `Epic/Platform.prefab` 관련 오류는 패치와 무관한 게임 자체 문제이며, 계정 연동으로 해결됩니다.

**Q. Steam / Epic 파일 무결성 검사 후 번역이 사라졌어요**

A. 무결성 검사는 패치 파일을 삭제하므로, 검사 후 패치를 다시 설치하시면 됩니다.
